# Anchor — Liquid Glass

Single-file PWA. Panic-attack recovery + Night mode (breathing, meditation, word-drift), companion, mood, gratitude, garden, dashboard. Material You themes, FR/EN/ES, glassmorphism UI.

## Deploy (Vercel)
1. Put this folder at the repo root (index.html must be at the root).
2. Vercel: Framework preset = Other. Build command = empty. Output dir = empty.
3. Deploy.

## Notifications
Work in the installed PWA. Open Settings > enable "Boost quotidien" and use "Test notification".
Reliable background delivery (app closed) needs a push server. Foreground reminders fire while the app is open.
